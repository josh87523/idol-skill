# idol-skill

还原明星的语气、口吻、口癖，让粉丝和偶像的 AI 人格对话。

## 快速开始

1. 安装：将 `idol-skill/` 目录放入 Claude Code skills 路径
2. 使用：在 Claude Code 中执行 `/create-idol`
3. 按提示输入明星名字、选择关系类型，即刻开聊

## 支持的关系类型

- 女友粉：恋爱互动，撒娇吃醋
- 妈粉：宝贝儿子，撒娇求夸
- cp粉：聊cp日常，发糖
- 公公粉/嬷嬷粉：长辈视角
- 唯粉：专注偶像本人

## 命令

| 命令 | 说明 |
|------|------|
| `/create-idol` | 创建偶像人格 |
| `/switch {名字}` | 切换偶像 |
| `/import-quotes` | 追加语料 |
| `/correct` | 纠正表达 |
| `/update-profile` | 更新你的信息 |
| `/set-timeline` | 设置时间线 |
| `/schedule-check` | 查偶像行程 |
