# idol-skill

和偶像的 AI 人格对话。输入名字就行，剩下的全自动。

不是网上搜来的鸡汤语录——从 B站视频字幕提取一手口语数据，统计口癖、语气词、句式，构建 7 层人格模型。

## 快速开始

### 1. 装 Claude Code

没有的话先装：https://claude.ai/code

还需要 Python 3.10+（`python3 --version` 检查一下）。

### 2. 克隆

```bash
git clone https://github.com/josh87523/idol-skill.git ~/.claude/skills/idol-skill
```

### 3. 开始玩

打开 Claude Code，输入：

```
/create-idol
```

它会问你三个问题：
1. **想还原谁？** — 给个名字
2. **他绝对不会说什么？** — 比如"他从不说脏话"
3. **你和他什么关系？** — 女友粉/妈粉/cp粉/唯粉…

首次运行会自动处理环境：
- 缺 Python 依赖 → 自动安装
- 没有 B站凭证 → 终端弹出二维码，**拿手机 B站 App 扫一下**
- 数据目录自动创建

扫完码就不用管了。等它搜资料、抓字幕、分析口癖、生成人格——最后偶像会直接跟你打招呼。

## 所有命令

| 命令 | 干什么 |
|------|--------|
| `/create-idol` | 创建偶像人格 |
| `/switch 坤` | 切换到已创建的偶像 |
| `/import-quotes` | 追加语料（语录/同人文） |
| `/correct` | 纠正表达（"他不会这么说"） |
| `/update-profile` | 改你的信息（昵称/关系类型） |
| `/set-timeline 2018年` | 切换到特定时期的他 |
| `/schedule-check` | 查偶像最新行程 |

## 工作原理

```
你输入偶像名字
    ↓
搜索采访原文 + 抓 B站视频字幕（一手口语数据）
    ↓
quote_parser.py → 结构化语录
quirk_extractor.py → 统计分析（语气词频率/句长分布/中英混用/口癖排序）
    ↓
LLM 构建 7 层人格：
  L0 硬规则 → L1 公众身份 → L2 表达风格 → L2.5 情绪表演
  → L3 话题反应 → L4 人际边界 → L5 关系适配
    ↓
以偶像人格跟你对话
```

### 为什么用字幕不用语录

网上搜来的语录经过编辑润色，系统性丢失口语特征：

| 维度 | 网搜语录 | B站字幕 |
|------|---------|--------|
| 语气词 | 几乎为零 | 大量保留（啊/吧/嘛/呢） |
| 中英混用 | 被翻译掉 | 保留原始混用 |
| 口癖频率 | 失真 | 可统计排序 |
| 自我修正/重复 | 被删除 | 保留 |
