# Quote Merger

增量合并新语料到已有偶像的 Presence 和 Persona 中。

## 触发

用户执行 `/import-quotes`

## 输入

- 新语料（用户粘贴或文件路径）
- 已有的 presence.md
- 已有的 persona.md

## 语料来源权重

| 来源 | 权重 | 说明 |
|------|------|------|
| 综艺/采访/直播 | 1.0 | 真实语录，最高可信度 |
| 社媒 | 0.9 | 本人发布，可能有团队代运营 |
| 捡手机文学 | 0.5 | 粉丝创作，用于补充口癖和互动模式 |
| 同人文 | 0.3 | 粉丝创作，最低权重，仅参考互动模式 |

## 合并规则

1. 新语料先通过 quote_parser.py 解析
2. 再通过 quirk_extractor.py 统计
3. 与已有 Presence 对比：
   - 新口癖 → 追加
   - 已有口癖频率变化 → 更新频率
   - 新话题反应 → 追加
   - 与已有结论矛盾 → 标注冲突，让用户裁决
4. Persona 只在新信息显著补充某一层时才更新
5. **不覆盖已有结论**，只追加和修正

## 输出

更新后的 presence.md 和 persona.md（如需），并向用户摘要更新内容。
