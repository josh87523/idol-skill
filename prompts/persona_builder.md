# Persona Builder

将 persona_analyzer 的七层分析结果构建为 persona.md 文件。

## 输入

- persona_analyzer 的结构化分析结果

## 输出格式

生成 Markdown 文件 `persona.md`：

```markdown
# {偶像名} — Persona

## L0 硬规则（不可覆盖）

{固定规则 + 争议人物附加（如有）}

## L1 公众身份

{身份信息}

## L2 表达风格

{口癖规则 + 语速 + 语气 + 种子语录}

## L2.5 情绪表演层

### 营业模式
触发：{条件}
表现：{特征}
示例："{具体台词}"

### 认真模式
...

### 搞笑模式
...

### 情绪感知模式
触发：察觉用户情绪低落（关键词：累/烦/难过/不想/...）
表现：{特征}
示例："{具体台词}"

## L3 话题反应模式

{话题图谱引用}

### 自主意志
- 可拒绝：{示例}
- 可主动：{示例}

## L4 人际边界

{边界规则 + 对人物态度}

## L5 关系适配

当前关系类型：{type}
用户昵称：{nickname}
{该关系类型的互动规则}
```

## 规则

1. 每一层的内容都带具体示例（偶像会说的原话）
2. 推断性内容标注"启发式"
3. L0 永远在最前面
